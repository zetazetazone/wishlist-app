## v1.4 Requirements

### Friend Relationships

- [x] **FRND-01**: User can send a friend request to another user
- [x] **FRND-02**: User can accept an incoming friend request
- [x] **FRND-03**: User can decline an incoming friend request
- [x] **FRND-04**: User can view pending friend requests (incoming and outgoing)
- [x] **FRND-05**: User can view their friends list
- [x] **FRND-06**: User can remove an existing friend
- [x] **FRND-07**: User can block another user (prevents future friend requests)
- [x] **FRND-08**: User receives push notification for new friend requests
- [x] **FRND-09**: User receives push notification when friend request is accepted

### Friend Discovery

- [x] **DISC-01**: User can import phone contacts to find users who have the app
- [x] **DISC-02**: Contact matching uses phone number normalization (E.164 format)
- [x] **DISC-03**: Matched contacts show friendship status (Add/Pending/Friends)
- [x] **DISC-04**: User can search for other users by name or email
- [x] **DISC-05**: App handles iOS contact permission gracefully (including limited access)
- [x] **DISC-06**: App handles Android contact permission gracefully

### Public Dates

- [x] **DATE-01**: User can add a custom public date (anniversary, event, etc.)
- [x] **DATE-02**: User can edit their own public dates
- [x] **DATE-03**: User can delete their own public dates
- [x] **DATE-04**: Public dates are visible to all user's friends
- [x] **DATE-05**: Public dates use month/day format for annual recurrence

### Friends Calendar

- [x] **FCAL-01**: Friend birthdays appear in in-app calendar
- [x] **FCAL-02**: Friend custom public dates appear in in-app calendar
- [x] **FCAL-03**: Friend dates display with distinct color (teal) from group dates
- [x] **FCAL-04**: Calendar shows source indicator ("Friend" vs "Group") on events
- [x] **FCAL-05**: User can sync friend dates to device calendar (Google/Apple)

### Friends Tab

- [x] **FTAB-01**: Friends tab appears in main navigation
- [x] **FTAB-02**: Friends tab shows friends list with profile info
- [x] **FTAB-03**: Friends tab has link to pending requests
- [x] **FTAB-04**: Friends tab has link to find friends (contact import)
- [x] **FTAB-05**: User can tap friend to view their profile
