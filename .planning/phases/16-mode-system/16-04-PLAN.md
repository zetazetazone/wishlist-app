# Plan 16-04: Fix GroupModeBadge Width (Gap Closure)

## Goal

Make GroupModeBadge compact (fit to content) instead of stretching to fill parent container width.

## Root Cause

`GroupModeBadge.tsx` line 31: The badge View has `flexDirection: 'row'` but no `alignSelf` constraint. When rendered inside GroupCard's `flex: 1` parent container, it inherits the full width.

## Tasks

### Task 1: Add alignSelf to GroupModeBadge

**File:** `components/groups/GroupModeBadge.tsx`

**Change:** Add `alignSelf: 'flex-start'` to the View style at line 31.

**Before:**
```tsx
<View
  style={{
    flexDirection: 'row',
    alignItems: 'center',
    backgroundColor: config.bgColor,
```

**After:**
```tsx
<View
  style={{
    flexDirection: 'row',
    alignItems: 'center',
    alignSelf: 'flex-start',
    backgroundColor: config.bgColor,
```

**Commit:** `fix(16): make GroupModeBadge compact (fit to content)`

## Verification

- Badge on GroupCard (home screen) should be compact around icon + text
- Badge should still render correctly in GroupViewHeader and settings contexts
